# 💬 Cord - Real-Time Chat

Cord is a real-time chat application inspired by Discord. Built entirely with Node.js, Socket.io, and Vanilla JS, it offers a seamless instant messaging experience complete with multi-channel support, live presence, and interactive message features.

**[🔴 Check out the Live Demo](https://cord.up.railway.app/)**

## ✨ Features

- ⚡ **Real-Time Messaging** - Instant delivery of messages across multiple connected users.
- 📂 **Multiple Channels** - Organize conversations by topic with dedicated chat rooms.
- 🟢 **Live Online Member List** - See exactly who is currently active in the app.
- ⌨️ **Typing Indicators** - Real-time "User is typing..." notifications.
- 😄 **Reactions & Emojis** - Full emoji picker and hover-to-react functionality on individual messages.
- 🔔 **Unread Message Badges** - Keep track of channels with new, unread activity.
- 🌗 **Dark / Light Mode** - Toggleable UI themes for comfortable day or night viewing.
- 🔊 **Sound Effects** - Audio cues for incoming messages and notifications.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Real-Time Communication:** Socket.io
- **Deployment:** [Railway](https://railway.app/)
