# Cord - Real-Time Chat

## Features
- Real-time messaging across multiple users
- Multiple channels
- Message reactions (hover a message to react)
- Emoji picker
- Dark / Light mode toggle
- Typing indicators
- Online member list
- Sound effects
- Unread message badges
